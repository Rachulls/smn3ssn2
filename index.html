LOS 4 CONCEPTOS MAS IMPORTANTES

Selectores

Los selectores permiten seleccionar elementos que se verán afectados por uno o más estilos.

  • Selector de tipo: Selecciona a todos los elementos de un solo tipo.
    Ejm: p{ color : red; }
  • Selector de clase: Selecciona a elementos que tiene atributo 'class'.
    Ejm: .ejemplo-clase{ color : red; }
  • Selector de ID: Selecciona a elementos que tiene atributo 'id'.
    Ejm: #ejemplo-id{ color : red; }

Herencia

Con herencia se refiere a la capacidad de los elementos hijos de heredar ciertos estilos de sus elementos padre.

  • body {color: blue;} Los elementos hijos del heredarán el color azul.

Cascada

En este caso las reglas son más específicas, el que aparece más abajo en el código tienen más peso y, por lo tanto, anulan reglas menos específicas o que aparecen antes.

Especificidad

La especificidad se refiere a la prioridad de una regla en función de su nivel de detalle.

  • Ejm de cascada y especificidad:
      • p { color: red; }
      .parrafo { color: blue; font-weight: bold; }
      #contenedor p { color: green; }
    1. La regla general p { color: red; } establece el color del texto de todos los elementos 'p' en rojo.
    2. La regla más específica .parrafo { color: blue; font-weight: bold; } establece el color del texto en azul y la negrita para los elementos con la clase "parrafo".
    3. La regla #contenedor p { color: green; } es aún más específica, ya que afecta a los elementos 'p' dentro del elemento con el id "contenedor" y establece el color del texto en verde.
